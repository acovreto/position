Css position

Css position property ovozmozuva bilo koj html element da se izvadi od normalnata pozicija i da se pozicionira na bilo koe mesto na veb stranata.

Vo CSS na selektiranite elementi mozeme da gi primenime sledinte tipovi na pozicioniranje:

  • position:static;
  • position:relative;
  • position:absolute;
  • position:fixed;
  • position:sticky;

Potoa elementot se pozicionira/izmestuva so dodavanje na nekoja vrednost na slednite svojtva/properties: top, right, bottom, left

**********************************************************************************

position:static;

  • elementite se po default static pozicionirani
  • se prikazuvaat na veb stranata po onaj redosled kako sto se definirani vo html fajlot
  • top, left ,bottom, right ne deluvaat(funkcioniraat)

**********************************************************************************

position:relative;

  • so position:relative bilo koj html element mozeme da go pomesteme od normaliot tek so definiranje na nekoe property:top, left, bottom, right
  • elementot se pomestuva vo odnos na negovata normalna pozicija, ili so drugi zborovi se pomestuva vo odnos na samiot sebe.
  • na mestoto na relativno pomesteniot element ostanuva prazen prostor
  • relativno pomesteniot element gi pokriva drugite elementi

Primer za position:relative;

Jas sum pozicioniran 'relative'

Za doma pozicioniraj me:

top:100px

left:300px

**********************************************************************************

position:abosolute;

  • position:absolute se koristi najmnogu
  • elementot se posicionira vo odnos na negoviot parent element
  • Treba da se zapomnat dve pravila
  • prvo:elementot sto se pozicionira treba da ima position:absolute
  • vtoro:parent elementot treba da ima position:relative

primer za position:absolute

Jas treba da sum pozicioniran relativno

Jas treba da sum pozicioniran absolutno

Za doma:Probaj da me poziconiras vo centar na parent elementot

**********************************************************************************

position:fixed;

  • position:fixed go pozicionira elementot relativno vo odnos na viewport/display
  • elementot ostanuva na istoto mesto i dokulku ja skrolame stranata
  • se posicionira so:top, left, right, bottom properies

Primer za position:fixed;

Za doma:pozicioniraj go button elementot vo desen dolen agol

**********************************************************************************

position:sticky;

  • position:sticky e koga nekoj element se dvizi pri skrolanje, a potoa se zastanuva ili zaglavuva na nekoja pozicija koja se definira so top, left, right, bottom properties.
